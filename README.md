# RatelData Website

Static landing page for RatelData - Enterprise Data Masking & Compliance Platform.

## Structure

```
webpage/
├── index.html          # Main landing page
├── css/
│   └── style.css       # All styles
├── js/
│   └── main.js         # Smooth scroll, animations
├── images/
│   └── README.md       # Placeholder for images
└── README.md           # This file
```

## Deployment

This is a static site designed to be hosted on GitHub Pages.

### Deploy to GitHub Pages:

1. Create a new repository (e.g., `rateldata-website`)
2. Push the contents of this `webpage/` directory to the repo
3. Go to Settings → Pages → Source: Deploy from branch (main)
4. Site will be available at: `https://<username>.github.io/rateldata-website/`

## Placeholders

The following sections have placeholders that need real content:

- **Hero Image**: Replace the placeholder with a platform screenshot
- **Demo Video**: Add a video embed (YouTube/Vimeo) or hosted MP4
- **Logo**: Add `images/logo.png`
- **Contact Email**: Update `contact@rateldata.com` with real email
- **Social Links**: Update GitHub, LinkedIn, Twitter URLs
- **Pricing**: Adjust plans and pricing as needed

## Dependencies

All loaded from CDN (no build step needed):
- Bootstrap 5.3
- Bootstrap Icons 1.11
